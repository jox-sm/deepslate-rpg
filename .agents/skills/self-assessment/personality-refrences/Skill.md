# Personality References Framework

A structured multi-lens personality framework that layers five distinct psychological models to produce a holistic, nuanced self-assessment. Each lens illuminates a different facet of personality — from core identity to stress behavior, unconscious patterns, and pathological expression.

## The Five Lenses (How It Works)

```
Lens 1: MBTI Core Type      →  Conscious identity & cognitive preferences
Lens 2: Grip/Stress States   →  Behavior under acute pressure
Lens 3: Enneagram            →  Core motivations, fears, and fixations
Lens 4: Shadow Functions     →  Unconscious / repressed cognitive modes
Lens 5: Dark Side            →  Pathological expression & character flaws
```

### 1. MBTI Core Type (Lens 1)
The baseline cognitive stack (dominant, auxiliary, tertiary, inferior functions). Determines how the individual naturally processes information and makes decisions.

### 2. Grip / Stress States (Lens 2)
When under extreme stress, each MBTI type "grips" — they regress to the unhealthiest expression of their inferior function. This accounts for out-of-character behavior and burnout patterns.

### 3. Enneagram Core (Lens 3)
Drives beneath cognition into motivation. The Enneagram type (1–9) reveals the core fear, desire, and fixation that energizes (or distorts) the MBTI expression. Essential for understanding *why* someone behaves, not just *how*.

### 4. Shadow Functions (Lens 4)
The unconscious cognitive stack (opposing, senex, trickster, demonic). These functions emerge in defensive, manipulative, or self-sabotaging behavior. Critical for blind-spot analysis and growth work.

### 5. Dark Side (Lens 5)
The crystallized pathology of each type — the characteristic way a personality breaks when defenses are permanently lowered. Useful for character design, conflict prediction, and identifying long-term derailment risks.

## How to Use This Skill

### Workflow

1. **Determine MBTI Core** — Use the MBTI reference to identify the four-letter type and its cognitive function stack.
2. **Assess Stress State** — Cross-reference the current situation against the grip-states reference to see which regressive patterns may be active.
3. **Identify Enneagram** — Match core motivations/fears from the Enneagram reference to add depth and driver analysis.
4. **Map Shadow Functions** — Use the shadow-functions reference to identify blind spots and unconscious behaviors that may emerge under conflict.
5. **Evaluate Dark Side** — Consult the dark-side reference to understand pathological tipping points and character-shadow material.

### Output Format

When producing a full assessment, structure the response as:

```yaml
core_type: ENTP
cognitive_stack: [Ne (dominant), Ti (aux), Fe (tertiary), Si (inferior)]
grip_state: Si grip — rumination, catastrophizing, rigid routine
enneagram: 7w8 — core fear: being trapped/pained; core desire: satisfaction
shadow_functions: [Ni (opposing), Te (senex), Fi (trickster), Se (demonic)]
dark_side: Gaslighting, manipulative debate, avoidance of emotional depth
```

For agent/charcater alignment, use the framework to:
- **Consistency check**: Ensure behaviors match the declared stack
- **Growth arc**: Plan development by integrating shadow functions
- **Conflict modeling**: Predict stress responses and dark-side triggers

## When Not to Use

- Do not use when a single-lens assessment (e.g., just MBTI) is sufficient
- Do not use for clinical diagnosis — these are personality frameworks, not psychiatric tools
- Do not treat any lens as deterministic; they are lenses, not cages
